# Verification Logs

This page serves as a central point for accessing and understanding the verification logs generated by the Autoflow system. These logs capture the outcomes of various verification processes, indicating successes and failures.

## Log Files

The verification logs are stored in the `.autoflow/logs/` directory. They are typically named following the pattern `verifier_<id>_<timestamp>_<status>.md`, where `<status>` is either `pass` or `fail`.

This directory contains a large number of log files, including:
- `verifier_..._fail.md`
- `verifier_..._pass.md`
- `verifier_idle_...md`

You can find detailed logs for specific runs in the `.autoflow/logs/` directory.
