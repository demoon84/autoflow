# desktop-runner-terminal-streaming

## Answer

데스크톱 runner terminal은 multi-line tick history를 타이핑 효과로 스트리밍하도록 개선됐고, runtime log를 바로 못 읽을 때는 단순 로그 파일 경로 문자열만 보여주지 않도록 fallback 처리됐다.
