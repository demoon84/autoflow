# merge-blocked-already-applied-patch-summary

## Answer

병합은 `dirty_scope_conflict`로 인해 차단되었는데, 이는 티켓 패치가 이미 적용되었지만 해당 파일에 관련 없는 변경 사항이 있었기 때문입니다. `git apply --check --reverse -`를 사용하여 패치 존재 여부를 확인하고, 이미 적용된 경우 해당 패치만 스테이징하여 병합을 완료할 수 있습니다.
