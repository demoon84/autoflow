# Wiki Source Summaries

This directory stores wiki-side summaries derived from `.autoflow/wiki-raw/`.

Each summary is generated by `autoflow wiki ingest <source-file>` unless `--no-summary` is passed. The summary frontmatter includes:

- `kind: source_summary`
- `slug`
- `created`
- `updated`
- `raw_source`
- `entities`
- `concepts`

Summaries cite their raw source file under `## Source`. Re-ingesting an unchanged source skips the adapter and preserves the existing summary byte-for-byte. Re-ingesting a changed source preserves `created:` and refreshes `updated:`.
