# Wiki 패널 - 좌우 분할 + 접이식 미리보기

## One-liner

Wiki 메뉴 레이아웃을 좌우 분할로 변경하고, 미리보기 패널을 접고 펼 수 있도록 개선합니다.

## Summary

Wiki 메뉴의 기존 상하 분할 레이아웃을 좌우 분할로 변경하여 사용성을 높입니다. 미리보기 패널은 기본적으로 닫힌 상태로 유지되며, 사용자가 목록에서 파일을 클릭할 때 자동으로 펼쳐지고, 명시적인 토글 버튼으로 다시 닫을 수 있습니다. 이는 항상 화면을 차지하던 비어 있는 LogPreview의 비효율성을 개선하기 위함입니다.

## Entities

- HandoffList (UI 컴포넌트): 핸드오프 파일 목록을 표시
- LogPreview (UI 컴포넌트): 미리보기 기능을 담당하며, 가시성과 위치가 변경됨
- WikiList (UI 컴포넌트): Wiki 파일 목록을 표시
- WikiQueryPanel (UI 컴포넌트): Wiki 검색 패널
- Wiki 메뉴 (UI 섹션): 설정 내 Wiki 섹션, 레이아웃 변경 대상
- apps/desktop/src/renderer/main.tsx (파일): Wiki 섹션 렌더링 및 상태 관리
- apps/desktop/src/renderer/styles.css (파일): 새로운 레이아웃 및 패널 스타일 정의
- isWikiPreviewOpen (React state): 미리보기 패널의 열림/닫힘 상태를 제어

## Concepts

- UI 문구 현지화: 사용자용 UI 문구는 한국어를 기본으로 사용
- 접이식 미리보기: 미리보기 패널이 필요할 때만 펼쳐지고, 기본적으로는 닫혀 있는 기능
- 좌우 분할 레이아웃: Wiki 메뉴의 주요 레이아웃 변경 방향
- 토글 버튼: 미리보기 패널을 수동으로 열고 닫는 데 사용되는 UI 요소
