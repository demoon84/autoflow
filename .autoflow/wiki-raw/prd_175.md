## Core Scope

- Goal: `tickets/inbox/order_160.md`의 요청을 planner runner wakeup 개선 작업으로 승격한다.
- In Scope:
  - planner loop runner 에 한정해 보드 입력 변경 시 즉시 다음 tick 으로 깨어날 수 있는 trigger file 계약을 추가한다.
  - 감시 대상은 `.autoflow/tickets/inbox/`, `.autoflow/tickets/backlog/`, `.autoflow/tickets/reject/`로 제한한다.
  - trigger burst 는 debounce 하거나 pending marker 1개로 병합해 planner tick 동시 실행을 막는다.
  - 기존 idle preflight skip (`planner_inputs_unchanged`) 과 동적 backoff 구현을 유지하면서, 입력 변경이 감지되면 extended sleep 을 조기 종료한다.
  - 이벤트 trigger 가 비활성화되거나 누락되어도 기존 interval / backoff 기반 polling 으로 회복한다.
  - 기능은 opt-in 으로 둔다: `AUTOFLOW_PLANNER_REALTIME_ENABLED=1`일 때 활성화하고, 기본값은 기존 동작과 호환되게 한다.
- Out of Scope:
  - worker / verifier / wiki runner 에 같은 실시간 trigger 를 확장하는 작업.
  - adapter prompt 내용, PRD/todo 생성 의미, reject auto-replan 정책 변경.
  - runner process kill / restart / OS process 관리.
  - 데스크톱 runner control 버튼 UX 변경.
  - `git push`.
