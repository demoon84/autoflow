# PRD 214 — SKILL 제거 Phase 3: 데스크톱 UI / IPC 제거

## Goal

데스크톱 앱의 "AI 스킬" 탭 UI, 관련 React 상태/헬퍼, preload/main 의 `controlSkill` IPC 채널, styles.css 의 `.skills-panel-*` 블록을 모두 제거한다. 이번 phase 이후 데스크톱 사이드바에 "AI 스킬" 진입점이 없어지고, renderer ↔ main 사이 skill 관련 IPC 도 사라진다.

## Background

- order_190 phase 3. phase 1/2 (prd_212/213) 가 백엔드 hook + CLI 표면을 제거한 뒤 데스크톱 표면을 떼어낸다.
- 직전 commit `11d7070` 이 desktop 에 AI 스킬 탭을 추가 — 사실상 그 변경의 revert 에 가까움.
- main.tsx: `controlSkill`, `loadSkills`, `viewSkill`, `archiveSkill`, `loadSkillHistory`, `SkillsPanel` 컴포넌트, `parseSkill*` 헬퍼, `settingsNavigation` 의 `skills` entry, `visibleSettingsSection === "skills"` 분기.
- styles.css: `.skills-panel-*` 블록.
- vite-env.d.ts / preload.js / main.js: `controlSkill` 타입 / API / IPC handler / `allowedSkillActions` set.

## Scope

- `apps/desktop/src/renderer/main.tsx`
  - state 변수 `skills`, `skillsLoading`, `selectedSkill`, `skillHistory`, `skillHistoryLoading` 등과 effect 들 제거.
  - `loadSkills`, `viewSkill`, `archiveSkill`, `loadSkillHistory`, `parseSkill*` 헬퍼 제거.
  - `controlSkill` 호출 site 모두 제거.
  - `SkillsPanel` 컴포넌트 제거.
  - `settingsNavigation` 배열에서 `skills` 항목 제거 (다른 항목 순서 유지).
  - `visibleSettingsSection === "skills"` 분기 제거.
- `apps/desktop/src/renderer/styles.css`: `.skills-panel-*` 로 시작하는 모든 selector 블록 제거.
- `apps/desktop/src/renderer/vite-env.d.ts`: `controlSkill` 타입 시그니처 제거.
- `apps/desktop/src/preload.js`: `controlSkill` API 노출 제거.
- `apps/desktop/src/main.js`: `autoflow:controlSkill` IPC handler 제거 + `allowedSkillActions` set 제거.

## Out of Scope

- `loadSkills` 외 다른 IPC 채널 (변경 없음).
- 데스크톱 데이터 모델 / 기타 settings 탭.
- AGENTS rule 본문 (phase 6).

- `apps/desktop/src/renderer/vite-env.d.ts`
