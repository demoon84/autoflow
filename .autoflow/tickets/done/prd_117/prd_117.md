# Project

- ID: prd_117
- Title: memo 흔적을 앱 전체에서 제거하고 quick-intake 를 order 로 통일
- AI: planner/worker
- Status: draft

## Core Scope

- Goal: Autoflow 의 quick-intake 채널 표기를 `order` 로 일관되게 통일하여 사용자 표면(라벨, 헤딩, CLI 서브커맨드, 파일명)과 내부 식별자(타입/함수/regex/변수) 모두에서 `memo` 의 흔적을 제거한다. 정책 문서는 더 이상 backward-compat 을 약속하지 않는다.
- In Scope:
  - CLI 서브커맨드 `autoflow memo create` → `autoflow order create`. 셸 스크립트 파일명 `packages/cli/memo-project.sh` → `packages/cli/order-project.sh`. `bin/autoflow` dispatch + help 텍스트 갱신.
  - 인박스 파일명 prefix `memo_NNN.md` → `order_NNN.md`. 신규 생성 로직, Plan AI 인박스 스캐너, 데스크톱 앱 regex/필터, runner 스크립트 모두 갱신.
  - 인박스 파일 본문 템플릿: `# Autoflow Memo` → `# Autoflow Order`, `## Memo` → `## Order`. 에러/도움말 메시지의 "Memo ..." → "Order ..." 또는 의미가 맞는 한국어 표현.
  - 데스크톱 앱(`apps/desktop/src/**`) 의 식별자/타입/함수/변수/라벨/문구 일괄: `TicketWorkspaceItemKind` 의 `"memo"` → `"order"`, `isMemoBoardFile` → `isOrderBoardFile`, `inboxMemos` → `inboxOrders`, `doneMemos` → `doneOrders`, `memoFiles` → `orderFiles`, `memoFilesById` → `orderFilesById`, `memoNumericId` → `orderNumericId`, `memoPinTitle` → `orderPinTitle`, regex `/^memo_\d+\.md$/i` → `/^order_\d+\.md$/i`, `workflowFileDisplayName` 의 `memo_` 분기 등.
  - 스킬 본문 정리: `.claude/skills/order/SKILL.md`, `.codex/skills/order/SKILL.md`, `integrations/claude/skills/order/SKILL.md`, `integrations/codex/skills/order/SKILL.md` 안의 backward-compat 설명 제거. AI 응답 가이드("memo 저장됨" 같은 표현)도 "오더" / "order" 로 정리.
  - 정책/문서/scaffold: `AGENTS.md`, `CLAUDE.md`, `.autoflow/AGENTS.md` (있으면), `.autoflow/agents/spec-author-agent.md`, `.autoflow/agents/plan-to-ticket-agent.md`, `.autoflow/rules/**`, `.autoflow/reference/**` (특히 `reference/memo.md` → `reference/order.md`), `scaffold/board/reference/memo.md` → `scaffold/board/reference/order.md`, `scaffold/board/AGENTS.md`, `scaffold/board/README.md`, `scaffold/host/AGENTS.md`, `scaffold/host/CLAUDE.md`, `README.md`, `packages/cli/README.md` 의 backward-compat 설명문 제거 + 일괄 order 표기.
  - 기존 데이터 마이그레이션: `.autoflow/tickets/inbox/memo_*.md` 와 `.autoflow/tickets/done/prd_*/memo_*.md` 를 `git mv` 로 `order_*.md` 로 rename (history 유지).
- Out of Scope:
  - 과거 git commit message / git history.
  - 이미 done 된 PRD/티켓 본문 안의 historical "memo" 언급 (단, `.autoflow/wiki/log.md` 의 새 항목은 order 표기).
  - `.autoflow/wiki/answers/**` 의 기존 answer 내용 (historical record).
  - 외부 문서, 외부 위키, 사용자가 별도로 작성한 노트.
  - 사용자 채팅 트리거 alias 추가 (`/memo` / `#memo` 는 부활시키지 않음).
