# PRD 215 — SKILL 제거 Phase 4: order/autoflow skill 의 RAG injection 제거

## Goal

`integrations/{claude,codex}/skills/{order,autoflow}/SKILL.md` 4 master 파일과 `.claude/skills/{order,autoflow}/`, `.codex/skills/{order,autoflow}/` 4 사본의 "Lookup Before Saving" 섹션에서 `autoflow wiki query --rag` 호출만 제거한다. `autoflow origin search` 호출은 그대로 유지한다 (Origin Ledger 는 별도 시스템).

## Background

- order_190 phase 4. SKILL infra 를 사용하던 trigger SKILL 들이 RAG 호출을 통해 SKILL 컨텐츠를 surface 했음. SKILL 폐기 후 RAG retrieval 자체가 무의미.
- Origin Ledger(`autoflow origin search`) 는 SKILL 과 무관하므로 보존.

## Scope

- `integrations/claude/skills/order/SKILL.md`
- `integrations/claude/skills/autoflow/SKILL.md`
- `integrations/codex/skills/order/SKILL.md`
- `integrations/codex/skills/autoflow/SKILL.md`
- `.claude/skills/order/SKILL.md`
- `.claude/skills/autoflow/SKILL.md`
- `.codex/skills/order/SKILL.md`
- `.codex/skills/autoflow/SKILL.md`

각 파일에서 `autoflow wiki query --rag` 가 들어 있는 "Lookup Before Saving" 단락(또는 동등 섹션) 의 wiki 호출 라인만 제거한다. `autoflow origin search` 라인과 다른 절차는 그대로 둔다.

## Out of Scope

- `/skill-this` SKILL.md (phase 5).
- 다른 trigger / skill 의 본문 변경.
- AGENTS rule 본문 (phase 6).
