## Core Scope

- Goal: `knowledge` 섹션의 Wiki 문서 미리보기 본문 렌더링을 `MarkdownViewer` 기반의 읽기 전용 viewer mode로 전환한다.
- In Scope:
  - Wiki 목록, handoff 목록, wiki query 결과에서 항목을 선택했을 때 열리는 문서/미리보기 본문을 raw `<pre>` 표시 대신 `MarkdownViewer`로 렌더링한다.
  - viewer mode는 읽기 전용이어야 하며 `textarea`, `contentEditable`, 편집 toolbar, 입력 커서가 노출되지 않아야 한다.
  - 기존 Wiki split-pane 흐름은 유지한다: 진입 시 미리보기 닫힘, 항목 선택 시 열림, 닫기 버튼과 "미리보기 열기" 토글 유지.
  - Markdown heading, list, code block, table, link 스타일은 기존 `.markdown-viewer` 스타일을 사용하되, Wiki preview pane 안에서 스크롤과 폭이 깨지지 않게 필요한 CSS만 보강한다.
  - `LogPreview`를 재사용한다면 Wiki 문서 viewer mode와 일반 로그 raw preview가 의도치 않게 섞이지 않도록 prop 또는 분기 기준을 명확히 둔다.
- Out of Scope:
  - wiki query CLI 동작, board file loading, selected path 상태 관리, Wiki 목록 필터링 변경.
  - 기존 Wiki preview open/close UX 재설계.
  - 로그/터미널/runner output을 Markdown viewer로 바꾸는 전역 변경.
  - 과거 `Todo-003` Wiki preview toggle 재시도나 `finish-ticket-owner` cleanup/smoke 출력 계약 수정.
  - MUI/Emotion 기반 신규 UI 추가.
