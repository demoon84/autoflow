## Core Scope

- Goal: 데스크톱 앱 왼쪽 사이드바의 주요 메뉴 항목 사이 세로 간격을 조금 넓혀, AI 진행 현황 / 티켓 / LLM 위키 / 통계 / 로그 항목이 더 여유 있게 보이도록 한다.
- In Scope: `.settings-nav-list` 또는 `.settings-nav-item` 스타일의 간격/패딩 조정, 활성 항목 하이라이트 박스와 hover/focus 상태가 자연스럽게 유지되는지 확인.
