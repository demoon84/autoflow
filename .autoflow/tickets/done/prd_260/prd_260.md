## Goal

runner 카드의 진행 단계 슬라이더에서 "진행 중" 단계와 "종료 결과"를 분리한다. cycle 흐름만 슬라이더에 표시하고, 종료 결과(완료/반려/오류)는 별도 배지/toast로 대체한다. planner/worker/wiki 3개 카드 각각에 역할에 맞는 단계 라벨을 적용한다.

## Background

- 현재 runner 카드 단계 슬라이더에 "진행 중" 단계와 "종료 결과"가 혼재해 의미가 불명확
- order_228(prd_259)의 runner-stage.js가 만드는 stage 값(planning/generating-todo/inprogress/merging/syncing/idle)과 UI 매핑 일치 필요
- 종료 결과(done/blocked/failed)는 슬라이더에서 제거해 cycle 흐름을 명확하게 표시
