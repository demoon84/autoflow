## Core Scope

- Goal: needs_user 또는 blocked 상태 진입 시 외부 채널(webhook, macOS notification, Slack)로 push 알림을 전송해 데스크탑 없이도 자율 운영 상태를 인지할 수 있게 한다.
- In Scope: `.autoflow/scripts/notify-user.ts` 신규(채널 어댑터: webhook URL POST, osascript, Slack webhook), `config.toml`에 `[[notifications]]` 블록, planner/worker의 needs_user 마킹 시 즉시 호출, blocked > `AUTOFLOW_NOTIFY_BLOCKED_THRESHOLD_SEC`(기본 1800) 시 호출, 동일 ticket 1시간 내 중복 발사 cooldown.
- Out of Scope: SMS/이메일 채널, 알림 내용 AI 요약, 원격 제어(resume 명령), 모바일 앱.
