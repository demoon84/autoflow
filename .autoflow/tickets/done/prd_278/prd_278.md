## Core Scope

- Goal: PTY mode 러너의 LLM 세션 누적 컨텍스트 비용 문제를 해결한다. 티켓 pass 직후 finalizer가 agent별 슬래시 명령(`/compact`, `/clear`, `/compress`)을 PTY에 자동 주입해 다음 티켓을 fresh-ish 컨텍스트로 시작하게 하고, 누적 토큰이 임계(`AUTOFLOW_CONTEXT_RESET_TOKEN_THRESHOLD`, 기본 100k)를 초과하면 full clear로 escalate한다.
- In Scope: `injectContextReset(runnerId, mode)` 헬퍼 구현, finish-ticket-owner pass 경로에서 트리거, env knob 3개, 로그 JSONL 1줄, ticket-owner-agent.md/AGENTS.md 정책 한 줄, runtime 미러, 회귀 테스트.
- Out of Scope: LLM 세션 완전 재설계, 새 슬래시 명령 발명, 인증/토큰 관리 변경.
