## Goal

Runner 카드(planner/worker/wiki)의 단계 indicator에서 점들을 잇는 track 선과 fill 선을 제거하고, 점(circle marker) 4개만 일정 간격으로 분리된 형태로 표시한다. 활성 단계 1개 highlight와 단계 라벨은 그대로 유지한다.

## Background

현재 progress indicator가 점 4개 + 연결선 + fill 선으로 progress bar처럼 보여 선형 진척을 암시한다. runner 흐름은 반복 cycle인데 progress bar 형태는 의미상 맞지 않는다. 시각적 무게를 줄이고 cycle 특성에 맞는 분리된 점 형태로 변경한다.

## Scope

- `apps/desktop/src/renderer/main.tsx`: track/fill 선 관련 렌더링 코드 제거
- `apps/desktop/src/renderer/styles.css`: `ai-progress-flow-track`, `ai-progress-flow-fill` 등 관련 CSS 룰 제거 또는 투명화

## Acceptance Criteria (Global)

- 점 4개 사이의 가로 연결선(track + fill) 시각적으로 사라짐
- 점 4개는 동일 위치에 유지, 활성 단계 1개 강조
- 단계 라벨(대기/구현/완료/반려 등) 유지
- planner/worker/wiki 3개 카드 모두 일관 적용
- 다크/라이트 테마 모두에서 잔여 배경선/shadow 없음
- 좁은 화면에서도 점 4개 layout 유지
