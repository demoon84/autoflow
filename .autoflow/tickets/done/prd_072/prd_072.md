- Title: 데스크톱 UI hover 툴팁 제거
- Goal: 데스크톱 앱의 아이콘 버튼 hover 시 뜨는 시각적 텍스트 말풍선을 제거해 러너 카드와 주요 툴바가 덜 거슬리게 보이도록 한다. 접근성 이름은 `aria-label` 로 유지한다.
- AI: planner
- Status: draft

## Core Scope

- Goal: `data-tooltip` 기반 커스텀 hover/focus 말풍선을 제거하되, 버튼의 동작과 스크린리더용 라벨은 유지한다.
- In Scope:
  - `apps/desktop/src/renderer/main.tsx` 에서 아이콘/컨트롤 버튼에 붙은 `data-tooltip` 속성을 제거한다.
  - 러너 카드/진행 보드의 시작, 중지 버튼과 상단/통계 영역의 새로고침, 설치, 지표 저장 같은 버튼에서 시각적 hover 말풍선이 더 이상 렌더링되지 않게 한다.
  - `aria-label` 은 유지해 키보드/스크린리더 접근성을 보존한다.
  - `title` 은 텍스트 overflow 설명이나 브라우저 기본 제목이 필요한 비버튼 정보 표시에는 유지할 수 있지만, 아이콘 버튼 hover 말풍선 제거 목표와 충돌하는 버튼성 요소에서는 제거하거나 `aria-label` 로 대체한다.
  - `data-tooltip` CSS 훅이 더 이상 사용되지 않으면 관련 스타일을 정리한다.
- Out of Scope:
  - 버튼 배치, 아이콘 종류, 러너 시작/중지 동작, disabled 조건 변경.
  - 프로젝트 경로, 최근 프로젝트, 로그 라인, wiki snippet 처럼 긴 텍스트 전체를 확인하기 위한 정보성 `title` 전면 제거.
  - shadcn/Radix 컴포넌트 구조 변경이나 데스크톱 화면 재설계.
