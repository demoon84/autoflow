## Global Rules

- Allowed Paths are relative to the host project root.
- Verification commands run from the host project root unless a ticket says otherwise.
- Keep acceptance criteria observable.
- `apps/desktop/src/main.js` is intentionally out of scope because active `Todo-139` / `prd_140` owns the `readBoard` stale-while-revalidate fallback path.
- `doctor`와 `metrics`가 동시에 실행될 때 서로 같은 heavy traversal을 증폭하지 않도록 shared cache, bounded recomputation, 또는 NOWAIT-style portable lock fallback을 사용한다.
- Cache fallback을 도입할 경우 stale 응답에는 `cache_status`, `stale`, `fallback`, 또는 동등한 key=value marker를 출력해 데스크톱과 로그에서 진단 가능해야 한다.
- 정상적인 단독 실행 성능은 유지한다. 두 명령의 단독 실행이 기존 1.5초 내외였다는 관찰을 악화시키지 않는다.
- `.autoflow/tickets/done`, `.autoflow/logs`, `.autoflow/telemetry/runs.jsonl`, `.autoflow/metrics/daily.jsonl` 같은 큰 입력은 매 refresh마다 중복 full scan하지 않는다.
