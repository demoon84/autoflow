## Core Scope

- Goal: `setupRequired` 상태의 진행 화면에서 `<h2>` 바로 위에 단독 렌더링되는 `FolderPlus` 아이콘만 제거한다.
- In Scope:
  - `apps/desktop/src/renderer/main.tsx` 의 `setup-required-panel` 섹션에서 제목 위 단독 `<FolderPlus className="h-6 w-6" aria-hidden="true" />` 렌더링을 제거한다.
  - 같은 패널의 제목, 설명, 설치 버튼, 설치 중 spinner, 버튼 내부 `FolderPlus` 아이콘은 유지한다.
  - `boardMissing` 과 `runnersUnconfigured` 두 setup-required 문구 모두 같은 패널을 쓰므로, 두 상태에서 제목 위 큰 아이콘이 보이지 않게 한다.
- Out of Scope:
  - 설치 버튼 내부 아이콘 또는 다른 화면의 `FolderPlus` 아이콘 제거.
  - `.setup-required-panel` 레이아웃 재설계, sidebar 동작 변경, 설치 flow 변경.
  - `apps/desktop/src/renderer/styles.css` 의 unused selector 정리. 이번 요청은 보이는 상단 아이콘 제거에 한정한다.
