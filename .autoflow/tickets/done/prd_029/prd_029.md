## Core Scope

- Goal: Reduce the desktop app's page-wide typography scale so the UI reads more compactly while preserving existing layout, controls, and readability.
- In Scope:
  - Adjust the shared desktop renderer typography foundation so normal page text, labels, metadata, and MUI-backed text render smaller in a consistent way.
  - Prefer centralized theme/CSS changes over scattered component-level font-size overrides.
  - Make only minimal component-specific adjustments if an existing hardcoded size prevents the page-wide reduction from applying cleanly.
- Out of Scope:
  - Do not redesign page layout, navigation, runner controls, ticket data loading, or board behavior.
  - Do not change user-visible copy except where a font-size-only style requires no text content changes.
  - Do not change product code outside the desktop renderer typography surface.
