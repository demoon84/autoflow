- In Scope:
  - `@media (max-width: 1279px)` 분기(line ~4019 ~4027) 안의 다음 규칙 제거:
    - `.ai-progress-config, .ai-progress-config-with-agent { grid-template-columns: repeat(2, minmax(0, 1fr)) !important; }`
    - `.ai-progress-config .runner-save-button { grid-column: 1 / -1; }`
  - `@media (max-width: 980px)` 분기(line ~6388) 안의 `.ai-progress-config-with-agent { grid-template-columns: repeat(2, minmax(0, 1fr)) !important; }` 규칙 제거.
  - `@media (max-width: 720px)` 분기(line ~6407 ~6414) 안의 다음 규칙 제거:
    - `.ai-progress-config, .ai-progress-config-with-agent { grid-template-columns: minmax(0, 1fr) !important; }`
    - `.ai-progress-config .runner-save-button { grid-column: auto; }`
  - 베이스 규칙(line 3181 / 3193 / 3209)의 `.ai-progress-config { grid-template-columns: minmax(0, 1fr) minmax(0, 1fr) auto !important; }`, `.ai-progress-config-with-agent { grid-template-columns: repeat(3, minmax(0, 1fr)) auto !important; }`, `.ai-progress-config .runner-save-button { ... }` 는 그대로 유지한다.
  - 좁은 viewport 에서 dropdown 글자가 잘릴 수 있어 placeholder/현재 값에 `text-overflow: ellipsis` 가 필요한 경우, 새 미디어쿼리를 추가하지 않고 기존 `.runner-select` 규칙(line ~3201)에 `text-overflow: ellipsis; white-space: nowrap; overflow: hidden;` 를 누적해 1줄 안에 자연스러운 절단이 일어나도록 보강한다.
- Out of Scope:
  - 같은 미디어쿼리 블록의 다른 셀렉터(`.ai-progress-row-top`, `.ai-progress-actions`, `.ai-progress-current*`, `.ai-progress-row-worker`, `.workspace-layout`, `.settings-page`, `.settings-nav*`, `.settings-content*`, `.metrics-strip`, `.runner-grid`, `.settings-content .runner-grid` 등)는 절대 변경하지 않는다.
  - `.ai-progress-board` 그리드(이미 prd_206 으로 1줄 3칸 고정 완료) 변경.
  - `apps/desktop/src/renderer/main.tsx` 또는 React 컴포넌트 구조 변경.
  - 새로운 미디어쿼리 추가 또는 viewport 단위(`vw`, `clamp`) 도입.
  - dropdown 본체 컴포넌트 동작 변경, 새 wrapper 추가, 가로 스크롤 컨테이너 신설.
