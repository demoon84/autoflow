## Core Scope

- Goal: 로그 페이지의 list pane 헤더에서 제목과 카운트가 두 줄로 분리되지 않고 같은 행에 표시되도록 JSX 또는 최소 CSS 를 조정한다.
- In Scope:
  - `apps/desktop/src/renderer/main.tsx` 의 로그 페이지 좌측 `section-heading compact` 헤더에서 `로그` 제목과 `최근 N / 전체 N건` 또는 `전체 N건` 카운트를 inline 구조로 정리한다.
  - 필요한 경우 `apps/desktop/src/renderer/styles.css` 에 로그 헤더 전용 또는 기존 `section-heading` 계열의 좁은 정렬 보정을 추가한다.
  - 카운트가 길어져도 제목, 카운트, `Terminal` 아이콘이 겹치지 않고 좁은 폭에서 자연스럽게 줄바꿈 또는 축약되도록 한다.
- Out of Scope:
  - 로그 목록 데이터, 필터/limit 동작, 로그 읽기 동작 변경.
  - Wiki 페이지나 작업 흐름 페이지의 다른 `section-kicker` + `<h3>` 헤더 일괄 재설계.
  - 데스크톱 화면 전체 레이아웃, navigation, runner lifecycle 로직 변경.
