- Goal: `apps/desktop/src/renderer/main.tsx` 의 `AiConversationPanel` 하단에 작은 한 줄 영역을 추가하고, 러너가 running 일 때만 elapsed time 과 누적 token 카운트를 표시한다. running 이 아니면 영역을 숨기거나 dimmed 로 둔다.
- In Scope:
  - AiConversationPanel 본문(narrative 요약본) 아래에 elapsed + tokens 한 줄 영역 추가.
  - elapsed 포맷: 60초 미만 `45s`, 이상 `12m 30s` 형식. 1초 주기 갱신 (setInterval 또는 requestAnimationFrame).
  - tokens: `runner.tokenUsage` 또는 telemetry stream 의 token_input/output 합산을 그대로 표시. 화살표 글리프(`↓` 등) 로 "수신" 의미 시각화.
  - running/idle/stopped 토글: 러너가 idle/stopped 일 때 영역 숨김 또는 dimmed.
  - `prefers-reduced-motion: reduce` 사용자에서 spinner 등 회전 애니메이션 비활성, 정적 dot 또는 텍스트만.
  - useEffect cleanup 으로 setInterval/RAF 누수 방지.
- Out of Scope:
  - 새 IPC 채널, preload bridge, main process 변경 (option B 의 raw stdout 터미널 임베드는 이번 PRD 대상이 아님 — 별도 PRD 후보).
  - PRD 207 의 헤더 binary 상태 인디케이터 변경 — 그 위 또는 아래에 별개 영역으로 둔다.
  - PRD 221 의 본문 타이핑 애니메이션 로직 변경 — 같은 패널이지만 본문 stream 영역과 충돌하지 않게 하단 별개 영역으로 분리.
  - 다른 패널(`runner-row`, `runner-console`) UI 변경.
  - 새 npm dependency 추가.
