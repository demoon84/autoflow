## Core Scope

- Goal: `apps/desktop/src/renderer/main.tsx` 의 `AiConversationPanel` 본문 (live adapter stream 박스) 에 글자 단위 타이핑 애니메이션을 추가한다. 새로 도착한 stdout chunk 는 글자 큐에 쌓이고 일정 간격으로 1~3 글자씩 DOM 에 release 한다. 큐가 임계 (예: 800 자) 를 넘으면 즉시 flush 로 catch-up.
- In Scope:
  - AiConversationPanel 본문 stream rendering 로직: 도착 chunk 를 글자 큐로 buffer, 타이밍은 setTimeout 또는 requestAnimationFrame 기반.
  - ANSI 색상 보존: ansi-to-html 으로 만들어진 color span 단위로 group 을 잘라낸 뒤 group 안에서 글자 release. color span 자체를 글자 단위로 쪼개지 않는다.
  - prefers-reduced-motion 미디어 쿼리 fallback: 애니메이션 disable, instant render.
  - 큐 길이 임계 catch-up flush.
  - useEffect cleanup 으로 setTimeout/RAF 누수 방지.
- Out of Scope:
  - AiConversationPanel 헤더 (binary 상태 인디케이터, 카드, 토큰 사용량 등) 변경.
  - 다른 패널 / `runner-console` / 다른 terminal 영역.
  - 새로운 IPC, 새 dependency 추가 (react-typing-effect 류 금지).
  - ANSI 파싱 자체 변경 — ansi-to-html 흐름은 그대로 둔다.
  - 자동 스크롤 / 사용자 스크롤-up pause 동작 자체를 바꾸는 작업 (기존 동작은 유지해야 한다).
