## Core Scope

- Goal: `AiProgressRow`의 카드 헤더 구조와 스타일을 좁게 조정해 러너 이름, 토큰 사용량, 시작/정지 버튼, 진행 트랙의 시각적 위계를 사용자가 요청한 2줄 구조로 맞춘다.
- In Scope:
  - `apps/desktop/src/renderer/main.tsx`의 `AiProgressRow` JSX를 조정해 첫 줄은 좌측 `아이콘 + 이름 + 토큰 사용량`, 우측 시작/정지 버튼으로 구성한다.
  - 진행 트랙(`대기 / 구현 / 완료 / 반려`)은 첫 줄 아래 별도 줄에서 카드 폭 전체를 사용하게 한다.
  - 토큰 사용량은 이름 옆 같은 inline 흐름에 배치하되, 긴 숫자도 레이아웃을 밀어내지 않도록 CSS에서 ellipsis 또는 안정적인 축소 동작을 보장한다.
  - `hideProgressTrack` 분기에서는 트랙이 없어도 첫 줄 정렬과 버튼 위치가 깨지지 않게 유지한다.
  - 기존 상태/메시지 영역(`ai-progress-current`)과 runner 설정/최근 터미널 출력 영역은 유지한다.
- Out of Scope:
  - runner 상태 계산, 토큰 사용량 집계, 시작/정지 동작 변경.
  - 진행 단계 이름, 색상, 상태 전이 로직 변경.
  - AI 관리/작업 흐름 페이지의 다른 카드 구조 변경.
  - 새 컴포넌트 라이브러리 도입 또는 전역 디자인 토큰 변경.
  - `finish-ticket-owner` runtime/smoke 출력 계약 수정.
