## Global Rules

- Allowed Paths는 프로젝트 루트 기준 상대 경로.
- `git push` 금지.
- 기존 즉시/force stop은 emergency path로 유지한다.
- drain-stop은 이미 잡은 작업을 중단하지 않는다. 새 claim/promotion만 막는다.
