# PRD 212 — SKILL 제거 Phase 1: Auto pipeline 차단

## Goal

Worker / runner 자동 SKILL 흐름을 코드 경로에서 분리한다. `finish-ticket-owner.sh pass` 시점의 `autoflow skill review --from-ticket` enqueue 호출과 worker tick 의 `AUTOFLOW_SKILL_NUDGE_*` 처리, skill review queue sidecar 파일을 모두 제거해 ticket 완료에서 skill 시스템으로 향하는 자동 호출이 0이 되도록 한다.

## Background

- order_190 사용자 결정: SKILL 시스템 전면 제거(option B). 단계별 회귀 위험 최소화를 위해 가장 먼저 자동 생성 파이프라인을 차단한다.
- 현재 hook: `.autoflow/scripts/finish-ticket-owner.sh` 의 pass 경로(rule 18a/18d) 가 `autoflow skill review` 큐 enqueue 를 best-effort 로 호출.
- 현재 nudge: `runtime/board-scripts/run-role.sh` + `packages/cli/run-role.sh` 가 worker tick 마다 `AUTOFLOW_SKILL_NUDGE_ENABLED` 분기로 nudge 를 emit.
- 이 phase 만으로도 자동 SKILL 생성 흐름이 완전히 멎는다. 후속 phase(213~217)가 CLI / UI / 컨텐츠를 정리.

## Decision

자동 호출만 차단한다. CLI(`autoflow skill ...`) 자체는 phase 2(prd_213) 에서 제거하므로 이번 PRD 에서는 호출 site 만 정리한다.

## Scope

- `.autoflow/scripts/finish-ticket-owner.sh`: pass 경로의 `autoflow skill review --from-ticket` 호출과 그에 딸린 evidence emit / 환경변수 분기(`AUTOFLOW_SKILL_AUTO_EXTRACT`, `AUTOFLOW_SKILL_REVIEW_*`) 제거. fail 경로는 변경 없음.
- `runtime/board-scripts/run-role.sh`, `packages/cli/run-role.sh`: skill nudge tick 코드 (`AUTOFLOW_SKILL_NUDGE_ENABLED`, `AUTOFLOW_SKILL_NUDGE_*` 분기) 제거.
- skill review queue sidecar 정리: `.autoflow/wiki/skills-local/.review-queue.tsv`, `.autoflow/wiki/skills-local/.usage.json` 이 존재하면 삭제(phase 6 에서 재정리될 것이지만 hook 제거 후 즉시 무용지물이므로 동시 제거).

## Out of Scope

- `bin/autoflow` 의 `skill)` case (phase 2).
- `packages/cli/skill-project.sh`, `runtime/board-scripts/curator-run.sh` (phase 2).
- 데스크톱 UI / IPC (phase 3).
- AGENTS.md rule 18a~18e 본문 정리 (phase 6).
- `.autoflow/wiki/skills/`, `.autoflow/wiki/skills-local/` 컨텐츠 / 디렉토리 (phase 6).

## Allowed Paths

- `.autoflow/scripts/finish-ticket-owner.sh`
- `runtime/board-scripts/run-role.sh`
- `packages/cli/run-role.sh`
- `.autoflow/wiki/skills-local/.review-queue.tsv`
- `.autoflow/wiki/skills-local/.usage.json`

- [ ] `runtime/board-scripts/run-role.sh` + `packages/cli/run-role.sh` 에 `SKILL_NUDGE` 가 0 hit.
- [ ] `.autoflow/wiki/skills-local/.review-queue.tsv` 와 `.usage.json` 이 존재하지 않는다.
