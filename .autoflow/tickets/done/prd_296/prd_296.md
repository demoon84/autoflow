## Global Rules

- Allowed Paths는 프로젝트 루트 기준 상대 경로.
- `git push` 금지.
- 새 parser를 중복 작성하지 말고 `board-utils.ts`를 우선 재사용한다.
- 단순 wrapper 추가가 아니라 `.sh` 본체 로직을 TS canonical로 축소/이관하는 것이 목표다.

## Global Acceptance Criteria

- [ ] 대상 보조 스크립트 중 bash 본체가 남아 있는 항목이 TS canonical 실행 경로로 축소된다.
- [ ] `.autoflow/scripts`와 `runtime/board-scripts` mirror가 대상 스크립트에 대해 동기화된다.
- [ ] `board-utils.ts`를 재사용하고 새 중복 parser를 만들지 않는다.
- [ ] 기존 CLI/output 계약과 exit code가 유지된다.
- [ ] shell/TS syntax check와 `./bin/autoflow guard . .autoflow`가 통과한다.
