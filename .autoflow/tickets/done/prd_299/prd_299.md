## Global Rules

- `run-role.sh`와 `runners-project.sh`는 고위험이므로 마지막 단계에서 진행한다.
- 기존 CLI usage/output 계약과 runtime mirror 관계를 유지한다.
- macOS bash 3.2 회피용 패치가 JS/TS 이관 과정에서 사라져도 동일 동작이 유지되어야 한다.
