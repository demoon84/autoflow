## Core Scope

- Goal: `apps/desktop/src/renderer/main.tsx` 의 AI progress board runner list 를 화면 렌더링 직전에 role 우선순위로 정렬해 `wiki` 와 `verifier` 위치가 바뀌어 보이지 않게 한다.
- In Scope:
  - `AIProgressBoard` 가 `board.runners` 에서 `self-improve` runner 를 제외한 뒤, 표시 전용 정렬 우선순위를 적용한다.
  - 정렬 우선순위는 `planner` / `plan` / orchestrator 계열, `ticket-owner` / `worker` / `ticket` 계열, `verifier` / `verification` / `veri` 계열, `wiki-maintainer` / `wiki` 계열, 기타 role 순서로 둔다.
  - 같은 우선순위 안에서는 기존 config 순서를 유지해 custom runner 의 상대 순서가 불필요하게 흔들리지 않게 한다.
  - `.autoflow/runners/config.toml` 의 runner 정의 순서는 바꾸지 않는다.
  - `apps/desktop/src/renderer/styles.css` 의 4-column desktop grid / 2-column / 1-column fallback 은 유지한다.
- Out of Scope:
  - runner start/stop/restart 동작, runner config schema, runner state storage 변경.
  - `.autoflow/runners/config.toml` 재정렬.
  - AI progress card 내부 UI, active ticket footer, token/log 표시, workflow pin strip 변경.
  - 신규 runner 추가 또는 legacy runner enable 정책 변경.
