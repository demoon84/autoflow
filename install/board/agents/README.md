# Agent Role Documents

Top-level files in this directory are active role contracts for the current
topology:

- `plan-to-ticket-agent.md` - planner runner
- `worker-agent.md` - worker runner
- `verifier-agent.md` - verifier runner
- `wiki-maintainer-agent.md` - wiki runner
- `spec-author-agent.md` - explicit PRD/spec authoring handoff

Deprecated role-pipeline contracts live under `legacy/`. They are installed for
old board/config compatibility only and should not be used for new runners.
