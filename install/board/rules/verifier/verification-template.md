# Verification Decision Template

Use this shape for verifier decision notes when a separate record is useful.

```md
# Verifier Decision: <Todo-NNN>

- Decision: pass|revise|replan
- Reason: <short concrete reason>
- Evidence:
  - <command, diff fact, file path, or observed output>
- Follow-up:
  - <none, worker revision request, or replan request>
```
