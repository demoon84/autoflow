# Raw Sources

이 디렉터리는 LLM Wiki로 반영할 원본 자료를 둔다.

원본 자료는 가능한 한 수정하지 않는다. 위키 러너는 이 자료를 읽고 `wiki/sources/`, `wiki/concepts/`, `wiki/decisions/`, `wiki/questions/` 아래에 정리된 markdown 위키 페이지를 만든다.

Obsidian Web Clipper를 사용하는 경우 clip 결과를 `raw/clipped/` 아래에 저장하면 된다. 민감 자료는 로컬 보드 안에 남기고, 공유가 필요할 때만 별도 문서로 명시적으로 꺼낸다.
