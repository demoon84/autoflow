# Agent Role Documents

이 디렉터리의 top-level 파일은 현재 topology의 active role contract다.

- `spec-author-agent.md` - `autoflow` skill 대화의 PRD 작성 계약
- `plan-to-ticket-agent.md` - 플래너 역할 assignment
- `worker-agent.md` - 워커 역할 assignment
- `wiki-maintainer-agent.md` - 위키 역할 assignment

`verifier-agent.md`는 legacy 호환 문서이며 현재 topology의 active role contract가 아니다.
